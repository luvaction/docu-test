
## 1. 에이전트 제어 방식

NetFUNNEL JavaScript 에이전트는 두 가지 방식으로 진입을 제어할 수 있습니다.

동적 제어는 URL 기반으로 제어하며, 넷퍼넬 콘솔의 세그먼트 설정에서 트리거룰을 통해 설정할 수 있습니다.

정적 제어는 에이전트에서 제공하는 함수를 통해 원하는 화면에 대기실을 적용할 수 있으며, 대기 진입 시점과 종료 시점을 제어합니다.

:::info
이 문서에서는 정적 제어에 대해 가이드합니다.
:::

---

## 2. 에이전트 적용

NetFUNNEL JavaScript 에이전트에서 제공하는 함수를 통해 특정 화면에 대기실을 적용할 수 있습니다.

:::info

넷퍼넬 함수에서 사용하는 projectKey, segmentKey는 **콘솔의 프로젝트 탭**에서 확인 가능합니다.
