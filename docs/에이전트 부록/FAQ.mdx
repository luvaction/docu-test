## Android

### 설치 및 초기 설정

<details>
  <summary>넷퍼넬 에이전트를 프로젝트에서 인식하지 않아요.</summary>

    프로젝트에서 (에이전트) .aar 파일이 제대로 인식되지 않았을 경우 발생합니다.

    1. Project 탭에서 libs/ 내 .aar 에이전트 파일에서 우클릭
    2. `Copy Path/Reference…` → `Path From Content Root`를 선택

    ![Copy Path/Reference…선택](/img/faq/android/android_faq_1.png)
    <small style={{ display: "block", textAlign: "center", color: "#666" }}>`Copy Path/Reference…`선택</small>

    ![Path From Content Root 선택](/img/faq/android/android_faq_2.png)
    <small style={{ display: "block", textAlign: "center", color: "#666" }}>`Path From Content Root` 선택</small>

    3. 복사한 경로를 build.gradle에 아래와 같이 등록 및 Sync Now 동기화
    ![Path From Content Root 선택](/img/faq/android/android_faq_3.png)
</details>

### 설정 옵션 이해

<details>
  <summary>넷퍼넬의 로그를 출력하고 싶어요.</summary>
    :::warning
    상용 서비스에서는 printLog=false로 설정해 로그 노출을 방지하는 것을 권장합니다.
    :::

    초기화 함수의 printLog 값을 true로 설정 시, Logcat을 통해 넷퍼넬 로그가 출력됩니다. “NetFUNNEL“ 태그로 출력되며, Logcat에서 package:mine NetFUNNEL을 필터링으로 확인 가능합니다.

    | 파라미터   | 타입     | 설명                                | 기본값 |
    |------------|----------|-------------------------------------|--------|
    | `printLog` | Boolean? | 넷퍼넬 디버그를 위한 로그 출력 유무 | false  |

    <Tabs>
        <TabItem value="Kotlin" label="Kotlin">
            ```kotlin
        import com.nf4.Netfunnel

        class SampleApplication : Application() {
            override fun onCreate() {
                super.onCreate()

                Netfunnel.initialize(
                    serverUrl = "{{SERVER_URL}}",
                    errorUrl = "{{ERROR_URL}}",
                    printLog = true
                )
            }
        }
            ```
        </TabItem>
         <TabItem value="Java" label="Java">
            ```java
        import com.nf4.Netfunnel;

        public class SampleApplication extends Application {
            @Override
            public void onCreate() {
                super.onCreate();

                Netfunnel.INSTANCE.initialize(
                        "{{SERVER_URL}}",
                        "{{ERROR_URL}}",
                        ...
                        true // printLog
                );
            }
        }
            ```
        </TabItem>
    </Tabs>

</details>

  <summary>대기실이 보이지 않아요.</summary>

</details>

## JavaScript

### 설치 및 초기 설정

<details>
  <summary>에이전트가 초기화되지 않아요.</summary>

    NetFUNNEL JavaScript 에이전트는 HTML의 `<head>` 영역에서 초기화하는 방식을 권장드립니다.
    JavaScript 내에서 초기화를 진행할 경우 환경 변수나 실행 시점 등 여러 요소에 따라 예기치 못한 동작이 발생할 수 있습니다.

    보다 안정적인 연동을 위해 HTML에서 초기화를 진행해주시기 바랍니다.
</details>
